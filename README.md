# Music Timer

指定した再生時間にぴったり収まるプレイリストを自動作成するアプリです。  
お気に入り曲やフォロー中のアーティストの楽曲から、指定した分数に合うようにプレイリストを作成できます。  

**https://musictimer.dpdns.org/**
