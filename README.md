# Music Timer

指定した再生時間にぴったり収まるプレイリストを自動作成するアプリです。
お気に入り曲やフォロー中のアーティストの楽曲から、指定した分数に合うようにプレイリストを生成します。

**https://musictimer.dpdns.org/**

## 主な機能

再生時間を分単位（3〜100分）で指定すると、プレイリストを自動作成します。
フォロー中のアーティストから選んで絞り込んだり、お気に入り曲だけでプレイリストを作ることもできます。
作成したプレイリストはアプリ内でプレビュー再生でき、Spotify / SoundCloud アプリで直接開くことも可能です。
